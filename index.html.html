O que é a Computação na Nuvem?

A Computação na Nuvem (Cloud Computing) é a entrega de serviços de computação — como servidores, armazenamento, bancos de dados, rede, software — através da internet (a nuvem). Em vez de possuir infraestruturas físicas, as empresas podem alugar acesso a qualquer coisa de que precisem, da infraestrutura ao software, em uma base de pagamento por uso.

Características da Computação em Nuvem

  • Escalabilidade: Recursos podem ser aumentados ou diminuídos conforme a demanda.
  • Elasticidade: Capacidade de adaptação a variações na demanda.
  • Modelo de Pagamento por Uso: Pagamento apenas pelos recursos consumidos.
  • Serviços sob demanda: Acesso a recursos e serviços quando necessário.
  • Automação: Gerenciamento e provisionamento automático de recursos.

Modalidades de Serviços

1. Infraestrutura como Serviço (IaaS)

  • Amazon Web Services (AWS)
  • Microsoft Azure
  • Google Cloud Platform (GCP)

2. Plataforma como Serviço (PaaS)

  • Heroku
  • Google App Engine
  • Microsoft Azure App Services

3. Software como Serviço (SaaS)

  • Google Workspace (G Suite)
  • Microsoft 365
  • Salesforce

Tipos de Nuvem

1. Nuvem Pública

A nuvem pública é oferecida por terceiros e é acessível ao público geral. Os recursos são compartilhados entre vários clientes.

2. Nuvem Privada

A nuvem privada é operada exclusivamente para uma única organização, proporcionando maior controle e segurança.

3. Nuvem Híbrida

A nuvem híbrida combina nuvens públicas e privadas, permitindo que dados e aplicativos sejam compartilhados entre elas.

Resumo dos Grandes Provedores de Cloud

AWS (Amazon Web Services)

A AWS é um dos maiores provedores de serviços de computação em nuvem, oferecendo uma ampla gama de serviços, incluindo computação, armazenamento e bancos de dados. É conhecida por sua robustez e escalabilidade.

Microsoft Azure

Azure é a plataforma de nuvem da Microsoft, que oferece soluções para computação, análise, armazenamento e redes. É bem integrado com outras ferramentas Microsoft e é amplamente utilizado em ambientes corporativos.

Google Cloud Platform (GCP)

GCP oferece uma série de serviços voltados para análise de dados, machine learning e infraestrutura. É conhecido por suas capacidades de análise e integração com outras ferramentas Google.
